##Ominous

Install node modules and start up [beefy](https://github.com/chrisdickinson/beefy) - a local development server designed to work with browserify:
```
npm install
```
(http://localhost:9966/)

Build project:
```
npm run build
```

Start beefy:
```
npm run server
```
